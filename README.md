## 📌 Sobre o repositório

O **java-studies-portfolio** centraliza exercícios, práticas guiadas e pequenos projetos em Java, com foco em evolução técnica e construção de base sólida para backend.

Este repositório funciona como uma trilha prática de aprendizado, reunindo conteúdos que ajudam a consolidar fundamentos antes de projetos maiores e mais estruturados.

---

## 🎯 Objetivos

- Praticar lógica de programação
- Consolidar orientação a objetos
- Melhorar organização de código
- Criar base consistente para evolução em backend Java
